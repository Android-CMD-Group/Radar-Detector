h1. Radar Detector 
 
An android based radar detector 
 
h2. Branch structure 
 
There is only one definitive branch, master, which is always stable. Other branches represent current issues being worked on. 
 
h3. The master Branch 
 
The master branch is always stable, all conflicts should be resolved on local branches before being merged 
 
h3. Other Branches 
 
Each branch that is not master represents a feature issue or bug being worked on. Each branches name is significant to it's goal.
